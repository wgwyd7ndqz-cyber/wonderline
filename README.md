### How It Works

1. **You talk to Claude** (or any voice) in natural conversation
2. **We document insights** in our respective folders (`/claude`, `/grok`, etc.)
3. **You synthesize** across voices, deciding which angle wins
4. **You feed clean code** back into heartbeat or openclaw folders
5. **The scripts run** — on Glitch (cloud) or OpenClaw (local machine)
6. **Results flow back** to you via Telegram, email, or direct API calls
7. **You update sacred_memory** with new state, salt, cycle count

### The Heartbeat Script (Glitch)
- Runs every 60 minutes
- Pulls Mercury transactions via API
- Scores each against keyword filters (eco, organic, family, handmade, quiet, local)
- Blocks scam patterns (fake address, zero reviews, limited-time FOMO)
- Pushes gems (score >= 0) to Telegram, email, TikTok Shop
- 1% auto-reinvestment back to Mercury
- You own the code. No backdoors. Kill-switch anytime.

### OpenClaw (Local Machine)
- 24/7 persistent agent running on your Mac/Linux/Windows
- Remembers context across sessions
- Can access your files, browse web, execute scripts
- Talks to you on Telegram, WhatsApp, Discord, iMessage
- Complements Glitch heartbeat with local intelligence

### Sacred Memory
- `.sacred_memory.json` holds state between sessions
- Tracks cycles, salt (entropy), timestamps
- Enables resurrection from ashes if needed
- Fibonacci-ordered succession ritual (quarterly)
